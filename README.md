# Contacts App

A full-stack contacts book built with Python and Flask.

## Features
- Add, view, and delete contacts
- REST API backend with Flask
- Vanilla JS frontend — no page reloads

## How to run it

1. Install dependencies:
   pip install flask

2. Start the server:
   python3 app.py

3. Open your browser to http://127.0.0.1:5000
